# The Best Books
## C/C++ programming & Bash scripting.

## The C Programming Language 1988
Mutloq klassika (O'tgan yillarning eng yaxshisi)
"The C Programming Language" — Brian Kernighan va Dennis Ritchie
Bu kitob dasturlash olamida shunchaki qo'llanma emas, balki "Injil" hisoblanadi. Dennis Ritchie — C tilining asoschisi.
Nima uchun o'qish kerak? Kitob juda qisqa, aniq va lo'nda yozilgan. C tilining falsafasini tushunish uchun undan yaxshiroq manba yo'q.
Darajasi: O'rta/Yuqori. (Yangi o'rganuvchilar uchun biroz qiyinlik qilishi mumkin).
Eslatma: Bu kitob ko'pincha "K&R" deb ataladi.

## C Programming: A Modern Approach
Zamonaviy va keng qamrovli (Eng yaxshi o'quv qo'llanma)
"C Programming: A Modern Approach" — K. N. King
Bugungi kunda universitetlarda va mustaqil o'rganuvchilar orasida eng tavsiya etilgan kitob.
Nima uchun o'qish kerak? Muallif C89, C99 va hatto C11 standartlarini qamrab olgan. Unda nafaqat til sintaksisi, balki loyihalarni qanday to'g'ri tashkil qilish ham o'rgatiladi.
Afzalligi: Mashqlar va savollar juda mukammal ishlangan.

## Modern C - Jens Gustedt
Professional darajaga chiqish uchun
"Modern C" — Jens Gustedt
Agar siz C tilining zamonaviy imkoniyatlaridan (C11, C17 standartlari) foydalanib, eng yuqori darajadagi xavfsiz va tezkor kod yozmoqchi bo'lsangiz, bu kitob siz uchun.
Nima uchun o'qish kerak? Bu kitob o'quvchini bosqichma-bosqich "darajalar"ga (levels) bo'lib o'rgatadi: Tanishish, Tajriba, va Mutaxassislik.

## Effective C
Effective C: An Introduction to Professional C Programming 
Xavfsizlik va chuqur bilim uchun
"Effective C: An Introduction to Professional C Programming" — Robert C. Seacord
Zamonaviy dasturlashda xavfsizlik (security) birinchi o'rinda. Bu kitob sizga professional darajada qanday qilib xatolarsiz va kiberhujumlarga chidamli kod yozishni o'rgatadi.


 ## Linux command line and Bash scripting
[The Linux Command Line" — William Shotts]
Nima uchun butun dunyodagi mutaxassislar aynan shu kitobni "yagona" deb hisoblashadi? 
Chunki u shunchaki quruq kodni emas, balki Bash'ning falsafasini o'rgatadi.
Ushbu kitobning 3 ta asosiy ustunligi:

### Fundamental bilim: 
Kitob sizni fayllar bilan ishlashdan boshlab, o'zgaruvchilar, 
sikllar va murakkab funksiyalargacha bosqichma-bosqich olib boradi.
Sifat va tushunarlilik: Muallif murakkab narsalarni juda sodda tilda tushuntirgan. 
Kitobni o'qish jarayoni xuddi tajribali ustoz bilan suhbatlashgandek tuyuladi.
Tizimlilik: Bash skript yozishda eng muhim narsa bu — mantiq. 
Bu kitob sizda "terminalcha fikrlash" qobiliyatini shakllantiradi.

### Qanday o'qish kerak?
Bu kitobni shunchaki o'qib chiqish kifoya qilmaydi. 
Eng yaxshi natija uchun:
Kitobdagi har bir misolni terminalda o'zingiz yozib, ishlatib ko'ring.
Skript yozish qismiga kelganda, kichik muammolaringizni hal qiladigan (masalan, fayllarni avtomatik arxivlovchi) kichik dasturlar tuzishga harakat qiling.
Bu kitobni o'qib tugatganingizdan so'ng, siz nafaqat Bash skriptlarini yozasiz, balki har qanday Linux tizimida o'zingizni "baliq suvda yurgandek" his qilasiz.
